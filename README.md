🚖 City Ride Analysis – Data Analytics Project
A comprehensive exploration of ride‑sharing performance, revenue, cancellations, and customer experience.

📌 Overview
This project analyzes a full year of ride‑sharing data, uncovering trends in bookings, vehicle performance, revenue, cancellations, and customer ratings.
The dataset spans Jan 1, 2024 – Dec 30, 2024  and includes over 150,000 bookings.

The goal of this project is to provide actionable insights for improving operational efficiency, customer satisfaction, and revenue optimization.
